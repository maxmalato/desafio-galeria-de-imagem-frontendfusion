# Gallery

## Sobre o Projeto

Gallery é uma aplicação desenvolvida com React e estilizada utilizando TailwindCSS, com o objetivo de exibir uma coleção de fotos de forma interativa e dinâmica. A aplicação consome dados de uma API para listar as fotos, permitindo que os usuários explorem, visualizem detalhes específicos e marquem suas imagens favoritas.

O design moderno e responsivo, combinado com animações atraentes da biblioteca TailwindCSS Animated e ícones do Boxicons, proporciona uma experiência de usuário envolvente.

## Funcionalidades

- Listagem de Fotos: Exibição de todas as fotos disponíveis na API.
- Favoritos: Permite marcar/desmarcar fotos como favoritas para fácil acesso posterior.
- Detalhes da Foto: Visualize informações adicionais de qualquer foto selecionada.
- Design Responsivo: Experiência otimizada para diferentes dispositivos.
- Animações Dinâmicas: Elementos animados para transições suaves e interatividade.

## Tecnologias Utilizadas

- React: Biblioteca JavaScript para construção de interfaces de usuário.
- TailwindCSS: Framework CSS para estilização rápida e customizável.
- TailwindCSS Animated: Biblioteca de animações integrada ao TailwindCSS.
- Boxicons: Ícones modernos para complementar o design da aplicação.
- API REST: Consumo de dados dinâmicos para exibição das fotos.
